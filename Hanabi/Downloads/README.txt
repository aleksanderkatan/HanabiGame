Here are downloads for our game.
The game was designed on linux and we encountered some problems for Windows.
If the jar file doesn't run, update your JDK (https://www.oracle.com/java/technologies/javase-jdk14-downloads.html)
The Windows doesn't have Purisa Bold font, so it is set to default and may be a little misplaced.
To play online you need to have public IP address or be in the same LAN. This can be easily done by using Hamachi (https://hamachi.en.softonic.com/)
